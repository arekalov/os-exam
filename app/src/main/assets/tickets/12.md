## 1. Что такое виртуализация?

**Виртуализация** — это технология, позволяющая создавать абстрактное (виртуальное) представление вычислительных ресурсов (процессора, памяти, хранилища, сети и т.д.), отделяя логическую структуру от физического оборудования.  

---

## 2. Основные типы виртуализации
### 1. Виртуальные машины (интерпретаторы)
- Обеспечивают выполнение программ в изолированной виртуальной среде, имитируя работу отдельного компьютера или виртуального процессора.
- Примеры: 
    - Java Virtual Machine (JVM)
    - Исполнение JavaScript в браузере
    - Интерпретатор Python

### 2. Контейнеры приложений
- Позволяют изолировать приложения и их окружение на уровне операционной системы, обеспечивая независимость и переносимость.
- Более легковесны, чем классические виртуальные машины.
- Примеры:
    - Docker
    - Solaris Containers
    - Linux Containers (LXC)

### 3. Аппаратная виртуализация
- Позволяет запускать несколько ОС (гостевых) на одном физическом сервере за счёт эмуляции аппаратных ресурсов.
- Примеры:
    - KVM
    - Hyper-V
    - VMWare
    - Virtual Box
- Также включает виртуализацию аппаратных устройств.

### 4. Облачные технологии
- Построены на основе аппаратной виртуализации.
- Дополнительно включают средства автоматического распределения ресурсов (provisioning), масштабирования, мониторинга, сервисные платформы.
- Примеры:
    - Amazon Web Services (AWS)
    - Microsoft Azure
    - Google Cloud Platform
